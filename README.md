# Nuxt.js Bolierplate

> Nuxt.js + Sass, Scss + Vuex + Axios + Google Analytics + Sitemap Auto Generator

## Build Setup

```bash
# install dependencies
$ npm run

# serve with hot reload at localhost:3001
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
