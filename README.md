## Build Setup

#### NPM

```zsh
# install dependencies
$ npm run

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start
```

#### Docker

```zsh
# serve with hot reload at localhost:3000
$ docker-compose up
```
